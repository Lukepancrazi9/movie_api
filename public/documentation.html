Movie API Documentation File

Application Description:

This web application provides users with access to information about different movies, directors, genres, and release dates
Users will be able to create an account, sign in, update their information, and create a list of their top movies.

Business Logic URL HTTP Method Request Body Format Respond Body Format
Return a List of All movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie containing title, year, genre, director. Example:
{ title: 'Ex Machina', year: '2015', genre: { genreType: 'Thriller', genreDescription: 'Thrillers are characterized and defined by the moods they elicit, giving their audiences heightened feelings of suspense, excitement, surprise, anticipation and anxiety.' }, director: { directorName: 'Alex Garland', directorBirth: 'May 26, 1970' } }
Return data about a genre by genre type /movies/genre/[genreType] GET None A JSON object holding data about a single genre containing a description. Example:
{ genreType: 'Thriller', genreDescription: 'Thrillers are characterized and defined by the moods they elicit, giving their audiences heightened feelings of suspense, excitement, surprise, anticipation and anxiety.' }
Return data about a director by name /movies/directors/[directorName] GET None A JSON object holding data about a single director containing director name and birth. Example:
{ directorName: 'Alex Garland', directorBirth: 'May 26, 1970' }
Allow new users to create an account /users POST A JSON object holding data about a new user to add, structured like:
{ "name": "Doo", "topMovies": [] } 		A JSON object holding data about a new user that was added, including a new ID. Example:
{ "name": "Doo", "topMovies": [], "id": "9834c98e-0b70-418f-84b4-09521c0833c9" }
Allow current users to update their information /users/[id] PUT A JSON object holding data about the current user to be updated, structured like:
{ "name": "Doo", "topMovies": [] } 		A JSON object holding data about the updated user. Example:
{ "id": 1, "name": "Doo", "topMovies": [] }
Allow users to add a movie to their Top Movies list /users/[id]/[movieTitle] POST None Text message indicating whether the user added the movie successfully.
Allow users to remove a movie from their Top Movies list /users/[id]/[movieTitle] DELETE None Text message indicating whether the user removed the movie from their Top Movies list successfully.
Allow current users to deregister /users/[id] DELETE None Text message indicating whether the user was removed successfully.
