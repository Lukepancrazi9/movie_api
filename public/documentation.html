Movie API Documentation File

Application Description:

This web application provides users with access to information about different movies, directors, and genres.
Users will be able to create an account, sign in, update their information, and create a list of their favorite movies.

Users

Business Logic URL HTTP Method Request Body Format Respond Body Format
Return a List of All users /users GET None A JSON object holding data about all users
Return of a User by Username /users/[Username] GET None A JSON object holding data about a single user, structured like:
{ "_id": "ObjectId", "Username": "String", "Password": "String", "Email": "String", "Favorites": ["ObjectId"], "Birthday": "Date" }
Allow new users to create an account /users POST A JSON object holding data about a new user to add, structured like:
{ "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date" } 		A JSON object holding data about a new user that was added, including a new ID. Example:
{ "_id": "ObjectId", "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date", }
Allow current users to update their information /users/[Username] PUT A JSON object holding data about the current user to be updated, structured like:
{ "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date" } 		A JSON object holding data about the updated user. Example:
{ "_id": "ObjectId", "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date" }
Allow users to add a movie to their Favorites /users/[Username]/movies/[MovieID] POST None A JSON object holding data about the updated user. Example:
{ "_id": "ObjectId", "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date", "Favorites": ["ObjectId"] }
Allow users to remove a movie from Favorites /users/[Username]/movies/[MovieID] DELETE None A JSON object holding data about the updated user. Example:
{ "_id": "ObjectId", "Username": "String", "Password": "String", "Email": "String", "Birthday": "Date", "Favorites": ["ObjectId"] }
Allow current users to deregister /users/[Username] DELETE None Text message indicating whether the user was removed successfully.
